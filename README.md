# Portfolio

This is a personal portfolio project. It includes:

- `index.html`: Main HTML file
- `style.css`: Stylesheet for the site
- `script.js`: JavaScript functionality
- `assets/`: Folder for images and other assets
- `assets/cv/`: Folder for CV or resume files

## Getting Started

Open `index.html` in your browser to view the portfolio.

## Customization

- Update `assets/cv/` with your latest CV or resume.
- Modify `style.css` to change the look and feel.
- Edit `script.js` to add or update interactive features.
